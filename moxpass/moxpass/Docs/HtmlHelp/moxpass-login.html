moxpass-login Manual Page

NAME

moxpass-login - Manage login behavior, set default email, reset password, and authenticated session.

SYNOPSIS

moxpass login [help | -h | --help] [<email>] [reset <email>]

DESCRIPTION

This command manages the current login session.

There are certain operations that can only be done with an authenticated login session, while others do not.

OPTIONS

-h
--help

Prints the synopsis and a list of commonly used commands.

MOXPASS-ACCOUNT COMMANDS

moxpass login [<email>]

Auth Required!

Login as an authenticated session based on the email and authenticated session settings in the setting file.

If the email argument is supplied, switch to that email temporarily and attempt to login. This does not change the email in the setting file.

moxpass login reset [<email>]

Enter password reset procedure based on the email in the setting file.

If the email argument is supplied, switch to that email and enter password reset procedure. This does not change the email in the setting file.
