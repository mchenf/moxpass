moxpass Manual Page

NAME

moxpass - A console application for managing local user's secrets and logins, written with C#.

SYNOPSIS

moxpass login|account|secret|help|config [<arguments>]

DESCRIPTION

This command is the main entrance to the different functions to moxpass application.

OPTIONS

-h
--help

Prints the synopsis and a list of commonly used commands.

MOXPASS COMMANDS

moxpass login [<arguments>]

Manages login functions, as well as reset. More details see: moxpass login.

Login is not required for most part. User will be prompt while accessing secrets. see How does encryption work in moxpass.

Once logged in, user can be in a state of authenticated session, where user can be exempt from entering a password.

moxpass account [<arguments>]

Manages functions related to accounts. More details see: moxpass account.

The command itself without any argument lists all the friendly names for the current logged-in account.

moxpass secret [<arguments>]

Manages secret functions, mostly secret generation functions. More details see: moxpass secret.
